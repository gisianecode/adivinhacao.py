# Jogo de Adivinhação

Um jogo simples de adivinhação feito em Python.

## Como funciona

O computador escolhe um número aleatório entre 1 e 100.  
Você deve tentar adivinhar qual é esse número, recebendo dicas:

- "O número é maior" ou
- "O número é menor"

O jogo continua até você acertar.
